Vamos a desarrollar una aplicación que reúna los indicadores de demografía, economía, turismo y conectividad aérea que se usan comúnmente en los estudios de mercado. La app debe tener una sección de indicadores a nivel Nacional, Estatal, Municipal, por Localidad y una selección de varias Ciudades importantes. Dado que las fuentes de datos para obtener los indicadores y gráficas requeridas son muchas y muy variadas, se requiere un ETL que guarde y procese los datos para que vivan en una base de datos (PostgreSQL), una API que consulte esos datos y los lleve al frontend.

La tipografía general de la app será personalizada (assets/ballingermono-light.ttf) y la paleta de colores de toda la app (que se irá aplicando a cada gráfica de forma progresiva dependiendo de la cantidad de elementos que requiera cada gráfica) será (#0576F3, #36F48C, #F47806, #F3F40B, #F479F4). La interfaz principal deberá tener en la parte superior izquierda un logo (assets/logo.png) y en seguida del logo, el título "Aplicación para estudios de Mercado". Debajo deberá mostrarse una sección para ajustes de estilo, que le permita al usuario modificar el tamaño de la tipografía (títulos, datos del eje x, datos del eje y) en las gráficas, cambiar entre la fuente por defecto o un set de fuentes de Google (que entre otras incluya la tipografía comfortaa), y una sección que le permita al usuario introducir una paleta de colores totalmente nueva (en formato HEX o seleccionar el color manualmente) o regresar a la paleta de colores por defecto.
